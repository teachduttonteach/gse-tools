  • getDataSheet(name?: string, sheetName?: string): SpreadsheetGS
    • Defined in DataSheet.ts:22

    Gets the data sheet for the current script. Uses the script ID to name the data sheet if it has not been created already.

    Parameters

    • name: string = 'gse-tools Settings'

      the name of the Spreadsheet to use; if 'id', then this function will use the script ID for the name of the sheet; otherwise will use 'gse-tools Settings' as the name of the sheet

    • Optional sheetName: string

      if only one sheet is desired, specify it here
